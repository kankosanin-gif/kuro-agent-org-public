# Phase 3 — KPI Framework

## KPI Domains

### 1) Planning Quality
- Requirement clarity score
- % requirements with acceptance criteria
- Requirement churn rate

### 2) Development Throughput
- Contract completion rate
- Lead time per contract
- Rework rate

### 3) Design Consistency
- Component reuse ratio
- UI deviation count
- UX friction issues per release

### 4) QA-QC Effectiveness
- Test coverage against requirements
- Defect escape rate
- Gate pass rate

### 5) DevSecOps Reliability & Security
- Deployment success rate
- Mean time to recover (MTTR)
- Critical vulnerability open count

### 6) Operations Stability
- Incident count
- SLA compliance rate
- Runbook coverage ratio

## Cadence
- Weekly manager KPI updates
- Bi-weekly cross-functional review
- Monthly Kuro-level strategy review

## Minimum KPI Report Schema
- Metric name
- Current value
- Previous value
- Delta
- Risk note
- Action owner
- ETA
