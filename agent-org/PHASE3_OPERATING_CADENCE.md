# Phase 3 — Operating Cadence

## Weekly Rhythm

### Monday — Planning & Risk Alignment
- Review requirement changes
- Confirm active contracts
- Prioritize gate-critical items

### Wednesday — Execution Sync
- Manager sync on blockers
- Cross-department dependency resolution
- Security/quality early-warning review

### Friday — Gate & KPI Review
- Gate status review (G1-G5)
- KPI delta review
- Next-week commitment freeze

## Monthly Rhythm
- Strategy review chaired by Kuro
- Standards conformance review
- Org-level process tuning decisions

## Standard Meeting Output
1) Decisions made
2) Risks accepted/escalated
3) Contracts opened/closed
4) Gate changes
5) Owner + ETA

## Failure Handling
- Any critical gate failure triggers hotfix planning cycle
- Repeated process failure triggers charter/prompt update
