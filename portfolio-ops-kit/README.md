# Portfolio Ops Kit (D)

A reusable operations kit for AI-agent engineering portfolios.

## Includes
- Issue templates for contract-driven execution
- Weekly review template
- Task contract template
- Hourly reporting guide

## Who this is for
- Solo builders scaling to manager/subagent operations
- Teams who need traceability and lightweight governance

## Quick Start
1. Copy `.github/ISSUE_TEMPLATE/*` into your repo
2. Use `templates/PHASE2_TASK_CONTRACT_TEMPLATE.md` for every task
3. Run weekly reviews using `templates/WEEKLY_REVIEW_TEMPLATE.md`
4. Add hourly reporting process from `docs/hourly-reporting.md`
